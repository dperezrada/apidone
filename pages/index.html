Create Resource

> curl -X POST "http://demo.apidone.com/movies" \
  -d '{"id": "123", "name": "Up", "year": "2009"}' \
  -H "Content-Type: application/json"
Status: 201
body: {"id":"123"}

List Resources

> curl -X GET "http://demo.apidone.com/movies"
Status: 200
body: [{"id":"123","name":"Up","year":"2009"}]

Create nested resource

> curl -X POST \
  "http://demo.apidone.com/movies/123/awards" \
  -d '{"name": "Oscar", "year": "2010", "total": 2}'
Status: 201
body: {"id":"509176c56768501c0c00018c"}
