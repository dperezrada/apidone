List resources

Method: GET

Example URL: /movies

Create resource

Method: POST

Example URL: /movies

Example POST Body: {"id": "123", "name": "Hot Fuzz"}

Update resource

Method: PUT

Example URL: /movies/123

Example PUT Body: {"id": "123", "name": "Hot Fuzz", "year": 2009}

Create resource (if id doesn't exists)

Method: PUT

Example URL: /movies/456

Example PUT Body: {"name": "Drunken Master"}

You can use any url to create a resource

Method: POST

Example URL: /my_url_is_using_prefix/tasks

Example POST Body: {"name": "Buy milk"}

Delete resource

Method: DELETE

Example URL: /movies/123

List with filter

Method: GET

Example URL: /movies?category=drama

Limit number of results

Method: GET

Example URL: /movies?_limit=10&_offset=20

List distinct elements of collection

Method: GET

Example URL: /movies?_select_distinct=name
