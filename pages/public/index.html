Examples

We can easily use APIdone to store, retrieve, update and delete information. For example we are going to show a simple way to store, movies information.

Create Resource

  • Description
  • jQuery
  • Curl
$.ajax({ url: 'http://demo.apidone.com/movies', type: 'POST', dataType: 'json', data: {"id": "123", "name": "Up", "category": "comedy"}, success: function(data, textStatus, xhr) { alert("Id:"+data.id); } });
curl -X POST "http://demo.apidone.com/movies" \ -d '{"id": "123", "name": "Up", "category": "comedy"}' \ -H "Content-Type: application/json"

List Resources

  • Description
  • jQuery
  • Curl
$.ajax({
  url: 'http://demo.apidone.com/movies',
  success: function(data, textStatus, xhr) {
    alert("Total movies: "+data.length);
  }
});
curl -X GET "http://demo.apidone.com/movies"

Get One Resource

  • Description
  • jQuery
  • Curl
$.ajax({
  url: 'http://demo.apidone.com/movies/123',
  success: function(data, textStatus, xhr) {
    alert("Movie name: "+data.name);
  }
});
curl -X GET "http://demo.apidone.com/movies/123"

Create Nested Resource

  • Description
  • jQuery
  • Curl
$.ajax({
  url: 'http://demo.apidone.com/movies/123/awards',
  type: 'POST',
  dataType: 'json',
  data: {"id": "1", "name": "Oscar", "year": "2010"},
  success: function(data, textStatus, xhr) {
    alert("Id:"+data.id);
  }
});
curl -X POST "http://demo.apidone.com/movies/123/awards" \
  -d '{"id": "1", "name": "Oscar", "year": "2010"}' \
  -H "Content-Type: application/json"

List Resources using filters

  • Description
  • jQuery
  • Curl
$.ajax({
  url: 'http://demo.apidone.com/movies?category=comedy',
  success: function(data, textStatus, xhr) {
    alert("Total movies: "+data.length);
  }
});
curl -X GET "http://demo.apidone.com/movies?category=comedy"
